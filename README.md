# React Reviews Web App 💻

[🔥 View it live here 🔥](https://react-reviews-ljfiv.netlify.app/)

React web app component to display customer reviews ⚛️

Features include:

- Hooks (useState and useEffect)
- Icons from React Icons / Font Awesome
